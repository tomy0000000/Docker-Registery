## Install

* Build the Dockerfile

```bash
docker build --tag tomy0000000/ubuntu:xenial .
```

* Start Container

```bash
docker run \
		-it --rm \
		--name="some-ubuntu" \
		--volume="$PWD:/home/$USER" \
		tomy0000000/ubuntu:xenial bash
```
